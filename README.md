> A modern, high-performance static documentation site for **Meu Livro 3**, built with VitePress and Vue 3 for a seamless reading experience.

## ✨ Features

- **Blazing Fast**: Static site generation powered by Vite.
- **Modern Layout**: Responsive and accessible design for all devices.
- **Asset Verification**: Custom scripts to ensure all images and docs are correctly linked.
- **Narrative Focus**: Optimized for long-form reading with clean typography.

## 🛠️ Tech Stack

- **Framework**: VitePress 1.6+
- **Frontend**: Vue 3 + TailwindCSS
- **Documentation**: Markdown-driven
- **Automation**: Node.js scripts for metadata management.

## 🚀 Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run docs:dev

# Build for production
pnpm run docs:build
```

## 🛡️ Antigravity Protocol (Documentation)

This project follows the **Antigravity** standards:
- **Modular Docs**: Chapters are strictly isolated in the `docs/` directory.
- **Clean Markdown**: Use semantic Markdown for better accessibility and AI context.
- **Asset Integrity**: Built-in verification loop for all media files.
